Configuration

All environment variables and configuration options for Bet4Gain.

Configuration is managed through environment variables in the .env file and config files in the config/ directory. Most settings can also be managed from the Admin Panel without touching any files.

Game Settings

File: config/game.php

Variable Default Description
GAME_MIN_BET 10 Minimum bet amount per slot
GAME_MAX_BET 100,000 Maximum bet amount per slot
GAME_MAX_PAYOUT_MULTIPLIER 1,000 Max payout multiplier cap
GAME_BETTING_DURATION 10 Betting phase duration in seconds
GAME_WAITING_DURATION 3 Wait time between rounds (seconds)
GAME_TICK_RATE 100 Multiplier broadcast interval (ms)
GAME_HOUSE_EDGE 0.03 House edge percentage (3%)
GAME_GROWTH_RATE 0.06 Exponential curve constant
GAME_DEMO_STARTING_BALANCE 10,000 Demo/guest starting coin balance
GAME_GUEST_PLAY_ENABLED true Allow guest/demo play
GAME_MAX_BETS_PER_ROUND 2 Max bets per round per user

WebSocket Configuration

File: config/reverb.php

Variable Default Description
BROADCAST_CONNECTION reverb Must be set to reverb for real-time game
REVERB_APP_ID — Reverb application ID
REVERB_APP_KEY — Reverb application key
REVERB_APP_SECRET — Reverb application secret
REVERB_SERVER_HOST 0.0.0.0 Server bind address
REVERB_SERVER_PORT 8080 Server listen port
REVERB_HOST localhost Client-facing host
REVERB_PORT 443 Client-facing port
REVERB_SCHEME https Client-facing scheme (http/https)
Important: The VITE_REVERB_* variables must match the Reverb client-facing settings. These are baked into the frontend JavaScript at build time, so you must run npm run build after changing them.

Caching Configuration

For production, use Redis for best performance:

# .env
CACHE_STORE=redis
SESSION_DRIVER=redis
QUEUE_CONNECTION=redis

REDIS_HOST=127.0.0.1
REDIS_PASSWORD=null
REDIS_PORT=6379

After making any changes, clear and rebuild the config cache: 

php artisan config:cache
php artisan route:cache
php artisan view:cache
