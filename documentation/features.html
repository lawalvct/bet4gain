Features Overview

A comprehensive breakdown of every feature in Bet4Gain.

🎮 Core Game

Real-Time Crash/Aviator Game

The centerpiece of Bet4Gain is a real-time multiplayer crash game. Each round follows a lifecycle:

  1. Waiting — Brief countdown between rounds (default: 3 seconds)
  2. Betting — Players place bets (default: 10 seconds)
  3. Running — Multiplier rises exponentially on a canvas-animated curve
  4. Crashed — Round ends at a random crash point; uncashed bets are lost

The multiplier follows the exponential formula e0.06 × t where t is elapsed seconds, broadcast to all players every 100ms via WebSockets.

Place Bet & Cash Out

  • Players can place up to 2 simultaneous bets per round in independent bet slots
  • Bets are placed during the betting phase with configurable min/max amounts
  • Manual cashout during the running phase locks in the current multiplier
  • Payout = bet amount × cashout multiplier

Auto Cashout

  • Set a target multiplier before placing a bet
  • The server automatically cashes out when the multiplier reaches the target
  • Works even if the player closes their browser — fully server-side

Auto Bet

Configure automated betting across multiple rounds with advanced strategies:

  • Base bet amount — Starting bet per round
  • Target multiplier — Auto-cashout point
  • Stop loss — Maximum total loss before auto-bet stops
  • Stop profit — Maximum total profit before stopping
  • Increase on loss — Percentage to increase bet after a loss (Martingale)
  • Increase on win — Percentage to increase bet after a win

Provably Fair

Every round outcome is cryptographically verifiable:

  • Server seed — Generated before the round, hash shared with players
  • Client seed — User-provided seed for added randomness
  • HMAC-SHA256 calculation determines the crash point
  • 1-in-33 chance of instant crash (1.00x multiplier)
  • 3% house edge applied mathematically
  • Dedicated verification page at /provably-fair

Game History

  • Recent crash results displayed in a scrollable rail above the canvas
  • Full paginated history page with crash points
  • Click any round to view detailed bets and verify fairness

💬 Live Chat

  • Real-time messaging via WebSocket presence channel
  • Emoji picker for rich expression
  • Player profile popovers — click any username to see stats
  • Online users list with live count
  • Win announcements — auto-broadcast when players cash out above a configurable multiplier (default: 10x)
  • Moderation tools — Admins can delete messages, mute/unmute users
  • Rate limiting — Configurable cooldown between messages

💰 Wallet & Payments

Dual Currency System

  • NGN Wallet — Real money (Nigerian Naira) for deposits and withdrawals
  • Coin Balance — Virtual currency used for betting
  • Exchange — Buy coins with NGN, sell coins for NGN with configurable rates

Payment Gateways

  • Paystack — Full integration with redirect-based checkout
  • Nomba — Alternative gateway with API-based flow
  • Webhook verification for both gateways
  • Configurable limits and fees

Withdrawals

  • Bank account resolution before withdrawal
  • Auto-approval for amounts below configurable threshold
  • Manual approval for larger amounts (via admin panel)
  • Configurable withdrawal fee percentage

P2P Coin Transfer

  • Send coins to other players by username
  • Gift or transfer modes
  • Configurable fee percentage (default: 2%)
  • Daily transfer limits
  • Minimum account age requirement
  • Real-time notification to recipient via WebSocket

🏆 Leaderboards

  • Four periods: Daily, Weekly, Monthly, All-Time
  • Rankings based on total winnings
  • Cached for performance (configurable TTL)
  • Auto-calculated via scheduled command
  • Public page at /leaderboard

👤 User Features

Authentication

  • Email/password registration and login
  • Social login via Google and GitHub
  • Email verification
  • Password reset flow
  • Two-factor authentication (2FA) support
  • Guest/demo play without registration

Profile

  • Upload custom avatar
  • Update username and email
  • Change password
  • User preferences/settings (sound, theme, etc.)

Statistics

  • Personal stats page: total bets, wins, losses, profit/loss
  • Bet history with filterable list
  • Transaction history with CSV export

📢 Advertisements

  • Admin-managed ad system with multiple placements:
    • Sidebar — Shown next to the game
    • Banner — Full-width banners
    • Popup — Modal advertisements
    • Between Rounds — Shown during waiting phase
  • Scheduling support (start/end dates)
  • Click and impression tracking
  • Automatic fallback when no ad matches a placement

🛡️ Security Features

  • Rate limiting — Per-action rate limits on all endpoints
  • Multi-account detection — Flag users sharing IP addresses
  • Suspicious activity monitoring — Auto-detect win streaks, rapid withdrawals, deposit spikes, bot patterns
  • IP blacklist/whitelist — Block or allow specific IP addresses
  • Security headers — X-Frame-Options, HSTS, CSP, XSS protection
  • Encrypted storage — Wallet balances and server seeds encrypted at rest
  • Audit log — All admin actions logged
  • Login tracking — IP, user agent, success/failure logging

🎯 Responsible Gaming

  • Self-exclusion — Players can exclude themselves for 1–365 days
  • Deposit limits — Daily, weekly, and monthly caps
  • Bet limits — Maximum bet amount override
  • Cooldown timer — Temporary lockout (15 min to 24 hours)
  • Session time reminders — Periodic alerts after configurable duration

📱 Responsive Design

  • Mobile-first layout with adaptive grid
  • Bottom navigation bar on mobile (Game, Chat, Bets, Leaderboard, Menu)
  • Touch-optimized bet controls and canvas
  • Tabbed interface on mobile to switch between panels
  • Three-column layout on desktop (Chat | Game+Bets | LiveBets+Leaderboard)

🌓 Theming

  • Dark and Light themes built with Tailwind CSS
  • User-togglable via header button
  • Respects system/OS preference by default
  • Preference persists across sessions

🔊 Sound Effects

  • Togglable sound effects for:
    • Bet placed
    • Cash out
    • Round crash
    • Chat message
    • Win notification
    • Countdown tick
  • Preference saved in user settings
