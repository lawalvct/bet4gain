Changelog

All notable changes to the Bet4Gain platform are documented here.

Versioning: Bet4Gain follows Semantic Versioning. Major versions may include breaking changes, minor versions add features, and patch versions fix bugs.

Version 1.0.0 — March 2026 (Initial Release)

The first public release of Bet4Gain — a complete, production-ready real-time multiplayer crash/aviator betting platform.

🎮 Game Engine

  • Real-time crash/aviator game with HTML5 Canvas rendering
  • Exponential multiplier curve with configurable growth rate
  • Up to 2 simultaneous bets per round
  • Auto cashout (server-side) with target multiplier
  • Auto bet with Martingale strategy support
  • Provably fair system with HMAC-SHA256 verification
  • Guest/demo play with virtual balance
  • Game history with per-round bet details

💰 Payments & Wallet

  • Dual currency system (NGN + Coins)
  • Paystack payment gateway integration
  • Nomba payment gateway integration
  • Coin exchange (buy/sell)
  • P2P coin transfers with fee system
  • Withdrawal approval workflow
  • Transaction history with CSV export

💬 Social

  • Real-time public chat with WebSocket presence
  • Emoji picker
  • Player profile popovers
  • Online users list
  • Win announcements
  • Chat moderation (mute/delete)

🏆 Leaderboards & Stats

  • Daily, weekly, monthly, all-time leaderboards
  • Personal statistics dashboard
  • Bet history

🛡️ Security

  • Rate limiting on all endpoints
  • Multi-account (IP) detection
  • Suspicious activity monitoring
  • IP blacklist/whitelist
  • Security headers
  • Encrypted wallet balances
  • Admin audit logging
  • Login attempt tracking

🎯 Responsible Gaming

  • Self-exclusion (1–365 days)
  • Deposit limits (daily/weekly/monthly)
  • Bet limits
  • Cooldown timer
  • Session time reminders

🛠️ Admin Panel

  • Filament-based dashboard with widgets
  • User management with ban/role controls
  • Game round inspection
  • Transaction management with approval workflow
  • Chat moderation
  • Advertisement management
  • Game, payment, and site settings pages
  • IP restriction management
  • Suspicious activity review
  • Reports and analytics

📱 UI/UX

  • Mobile-first responsive design
  • Dark/light theme with system preference detection
  • Sound effects (togglable)
  • Toast notifications
  • Social login (Google, GitHub)
  • Two-factor authentication support
